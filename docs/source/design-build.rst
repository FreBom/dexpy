Building a Design
=================

First you will need to determine what kind of design suits your problem.

=================
Screening Designs
=================

These designs are meant to be run with many factors, and are used to determine which factors are important
and which can be discarded. They typically have fewer runs and are meant to be used with a linear or interaction model.
